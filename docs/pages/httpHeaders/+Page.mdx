**Environment**: client.

Send additional HTTP headers to be sent along Telefunc HTTP requests.

Usually used for sending authentication headers.

```js
// Environment: client

import { config } from 'telefunc/client'

config.httpHeaders = {
  Authorization: `Bearer ${token}`
}
```
